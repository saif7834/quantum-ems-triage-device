# Quantum-Inspired Optical EMS Triage Device

## 🚀 Overview
This project is a low-cost optical sensing system designed for emergency medical triage. It measures vital parameters such as heart rate (HR), SpO2, respiratory rate (RR), and perfusion index.

## 🔧 Features
- Real-time HR, SpO2, RR monitoring
- Signal processing for noise reduction
- Balanced detection and coded illumination
- OLED display integration
- BLE communication

## 🧠 Technologies Used
- Embedded Systems (ESP32 / STM32)
- C/C++ (Firmware)
- Python (Signal Processing)
- OpenCV

## 📊 Performance
- Improved signal accuracy under noisy conditions
- Real-time monitoring system

## 🔌 Hardware Components
- ESP32 / STM32
- Pulse Sensor / Photodiodes
- OLED Display
- Power Module
